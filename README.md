# 100 Days of Code - Python

Welcome to my repository for the "100 Days of Code" challenge, a Python course by Angela Yu on Udemy. This repository contains all the files, projects, and notes I’ve completed throughout the course, which focuses on building Python programming skills through practical exercises.

## About the Course

The "100 Days of Code" challenge is designed to help learners become proficient in Python by dedicating at least 1 hour a day for 100 days. The course covers a wide range of Python topics, from beginner to advanced, and introduces concepts such as programing logic, data structures, web development and more.

## Course Details

- **Instructor:** Angela Yu
- **Platform:** Udemy
- **Duration:** 100 days
- **Main Language:** Python
- **Key Topics Covered:**
  - Python basics and syntax
  - Data structures (lists, dictionaries, sets, tuples)
  - Object-Oriented Programming (OOP)
  - Automation with Python
  - And much more...

## Repository Structure

This repository is organized by each day of the course, with each folder containing the corresponding project, code, and notes for that day.

- `Day 1-100/` - Each folder corresponds to a specific day’s work.
- `Projects/` - Completed projects and exercises from the course.
- `Notes/` - A summary of key concepts covered each day, with explanations and additional resources.
